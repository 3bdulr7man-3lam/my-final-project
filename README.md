# Simple Interest Calculator

## Project Description
A simple interest calculator built using a Bash script. 
It takes user inputs such as principal amount, rate of interest, 
and time period to calculate the simple interest.

## Simple Interest Formula
Simple Interest = (Principal × Rate × Time) / 100

## Inputs
- **Principal** – The initial amount of money
- **Rate of Interest** – The annual interest rate (in %)
- **Time Period** – The duration (in years)

## Output
- **Simple Interest** – The calculated interest amount

## Usage
Run the Bash script:
```bash
bash simple-interest.sh
```
